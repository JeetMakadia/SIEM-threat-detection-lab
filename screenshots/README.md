## How to Take Screenshots

### Windows VM
- Press Win + Shift + S to capture a region
- Or use Snipping Tool (search in Start menu)
- Save as PNG

### From Host Machine
- Use your host OS screenshot tool
- Capture the browser showing Splunk UI

## Naming Convention
Use descriptive names with underscores, all lowercase.
Example: brute_force_21_events_detected.png
